Vereinzelt können Dateien in der Dateiablage derzeit nicht geöffnet werden. Es erscheint folgende Fehlermeldung: "Beim Öffnen der Datei ist ein Fehler aufgetreten. Drücken Sie OK um den Editor zu schließen."
Dies betrifft sowohl das Öffnen von Dateien via öffentlichem oder internem Link als auch direkt aus der Dateiablage heraus. Hierbei lässt sich bislang kein reproduzierbares Muster erkennen, wann welche Dateien betroffen sind. 

So können Sie wieder auf die Datei zugreifen: Laden Sie die Datei herunter und laden Sie anschließend exakt die gleiche Datei wieder hoch. Hierdurch ist der Zugriff und das Bearbeiten wieder möglich, ohne dass hierbei bspw. der Freigabelink verändert wird.
